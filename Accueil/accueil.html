- Apprenez à développer vos premiers sites WEB :


Bienvenue sur mon site, vous y trouverez plusieurs cours sur les langages indispensables à la création d'un site. Vous apprendrez pas à pas les bases du développement WEB.



Les différents langages et leurs utilités :


Le HTML5 :


HTML signifie «HyperText Markup Language» qu'on peut traduire par «langage de balises pour l'hypertexte». Il est utilisé afin de créer la structure même d'une page WEB. Ce langage utilise des balises spécifiques pour indiquer au navigateur ce qu'il doit afficher. Le HTML est aujourd'hui dans sa 5ème version, on l'indique avec HTML5.


En suivant le cours dédié au HTML :


Vous apprendrez pas à pas à créer votre première page WEB, à y ajouter des titres, des paragraphes, des images et d'autres éléments en utilisant les balises adaptées.



Le CSS3 :


Le CSS (Cascading Style Sheets en anglais, ou «feuilles de style en cascade») est le code utilisé afin de mettre en forme une page web. Toute la partie de présentation, de WEB-design se fait grâce au CSS. Ce langage permet d'améliorer grandement l'affichage de vos pages HTML.


En suivant le cours dédié au CSS :


Vous apprendrez pas à pas à modifier la couleur de votre texte, ainsi que celle de l'arrière-plan. Le texte pourra désormais être stylisé. Vous pourrez insérer une image en arrière-plan et faire des animations d'un élément HTML, et bien d'autres choses...



Le Javascript


Le Javascript est un langage de programmation permettant de rendre vos pages WEB dynamiques : vos pages pourront désormais réagir aux interactions de l'utilisateur. Ce langage étend grandement les possibilités de création d'un site.


En suivant le cours dédié au Javascript :


Vous apprendrez pas à pas à afficher un message dans une boite de dialogue au chargement de votre page, à reconnaitre la touche du clavier pressé par l'utilisateur ; ainsi qu'à faire un système de mot de passe.
