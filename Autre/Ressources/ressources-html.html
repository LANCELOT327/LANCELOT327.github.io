Les Balises HTML

Les balises HTML permettent d'afficher des éléments sur notre page, et de créer la structure minimale de la page. On a donc besoin de les utiliser souvent : c'est pourquoi je vous ai fait un tableau de rappels avec la plupart des balises.

Balises Utilisations Exemples
<p>...</p> Permet d'afficher un paragraphe <p>ceci est un paragraphe</p>
<h1>...</h1> ou <h6>...</h6> Permet d'afficher un titre. Un nombre entier compris entre 1 et 6 est à indiquer après "h" : h1 étant le titre le plus plus important et h6 le titre le moins important. <h1>ceci est un titre</h1>
<img src="..." alt="..."> Permet d'afficher une image. L'emplacement est à renseigner dans src, une description brève est à renseigner dans alt. <img src="../../Images/Logo/image.png" alt="image" />
<a href="...">...</a> Permet d'afficher un lien hypertexte. L'URL ou l'emplacement de la page est à renseigner dans href. Le second espace doit contenir l'élément cliquable (mot ; image...) <a href="www.google.com">Ceci est un lien vers Google</a>
<table>...</table>

<tr>...</tr>

<td>...</td> 		L'élément <table> permet d'afficher un tableau. L'élément HTML <tr> définit une ligne de cellules dans un tableau. Une ligne d'un tableau est composée d'éléments <td>, qui constituent les données des cellules. <table>
